# 🏜️ 沙海淘金

一款沙漠生存 Roguelike 浏览器游戏——从营地出发，穿越沙漠节点地图，活着挖到金矿并返回。

### 怎么玩

1. 用浏览器打开 `index.html`（或访问部署后的网址）
2. 选择角色和难度，购买物资，出发！
3. 鼠标点击操作，管理水、食物、HP、体力，应对随机事件
4. 到达金矿挖金后返回营地即为胜利

### 资源与操作

- **移动**：每步消耗 1💧 1🍖 1⚡，点击相邻节点前进
- **休息**：消耗 1💧 1🍖，恢复体力（绿洲 +1.5，沙漠 +0.5，沙暴 -0.5）
- **道具**：水袋、干粮、药品、骆驼、帐篷、指南针、净水片、燃油
- **事件**：到达节点随机触发好/坏/中性事件
- **声望**：成功返回结算声望，解锁新角色和道具

### 角色

| 角色 | 技能 | 解锁 |
|------|------|------|
| 🧑 探险家 | 标准属性 | 初始 |
| 👩‍🔬 地质学家 | 挖金收益 +30% | 240⭐ |
| 🧕 游牧民 | 水粮消耗 -20% | 300⭐ |
| 👨‍🚀 老兵 | HP +30，战斗必胜 | 360⭐ |
