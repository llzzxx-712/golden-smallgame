# 沙海淘金 - 游戏设计文档

> 日期：2026-06-19 | 状态：设计完成

## 概述

一款单人 Roguelike 节点式路线生存游戏。玩家从营地出发，在沙漠节点地图上规划路线，管理水/食物/体力/血量等资源，活着到达金矿挖金并返回营地。

- **风格：** 扁平卡通，纯 CSS + emoji，零图片资源
- **技术：** 纯前端 HTML/CSS/JS，单文件架构，localStorage 存档
- **平台：** 浏览器打开即玩，无需服务器

---

## 1. 核心循环

```
选择难度 → 营地出发 → 购买物资 → 选择路线节点前进
    → 节点触发事件 → 资源消耗/补给 → 继续前进
    → 到达金矿 ⛏️ → 挖金 → 返程
    → 回到营地 🏕️ = 胜利
    → 水/食物/HP 归零 = 死亡
```

## 2. 资源系统

| 资源 | 用途 | 归零后果 |
|------|------|----------|
| 💧 水 | 每步消耗 | 渴死 |
| 🍖 食物 | 每步消耗 | 饿死 |
| ❤️ HP | 遇险受伤扣减 | 死亡 |
| ⚡ 体力 | 限制单次可走步数，扎营恢复 | 无法移动 |
| 💰 金币 | 起始发放，买物资，挖金赚取 | 无钱买补给 |

## 3. 难度系统

| 参数 | 简单 | 普通 | 困难 |
|------|------|------|------|
| 绿洲数量 | 较多 | 中等 | 稀少 |
| 好事件权重 | +10% | 不变 | -10% |
| 起始金币 | 150 | 100 | 60 |
| 沙暴节点 | 少 | 中 | 多 |
| 声望倍率 | ×1.0 | ×1.2 | ×1.5 |

## 4. 节点类型

| 节点 | 功能 |
|------|------|
| 🏕️ 营地 | 起点/终点，可买卖物资 |
| 🌵 沙漠 | 普通节点，触发随机事件 |
| 💧 绿洲 | 免费补水，好事件概率 +20% |
| 🏚️ 废墟 | 探索可得金币/道具 |
| 🐪 商队 | 买卖物资（价格±浮动） |
| ⛏️ 金矿 | 终点，挖金 |
| 🌪️ 沙暴 | 危险节点，坏事件概率 +40% |

- 地图每局随机生成，保证至少 1 条营→金矿通路
- 每条连线消耗 1 体力 + 1💧 + 1🍖
- 初始资源：HP=100, 体力=5, 水=5, 食物=5
- 到达金矿挖得 80~120 随机金币（地质学家 +30%）
- 起始金币按难度：简单 150 / 普通 100 / 困难 60

## 5. 道具系统

| 道具 | 效果 | 价格 |
|------|------|------|
| 💧 水袋 | +3 水 | 15 |
| 🍖 干粮 | +3 食物 | 15 |
| 💊 药品 | +30 HP | 25 |
| 🐪 骆驼 | 每步体力消耗减半（永久） | 50 |
| ⛺ 帐篷 | 扎营，恢复全部体力 | 30 |
| 🧭 指南针 | 显示相邻节点类型 | 20 |
| 💠 净水片 | 在绿洲额外 +2 水 | 10 |
| ⛽ 燃油 | 载具移动 3 步不耗体力 | 20 |

购买时机：营地出发前、商队节点。

## 6. 随机事件

### 好事件

| 事件 | 效果 |
|------|------|
| 发现水源 | 💧 +2 |
| 仙人掌果实 | 🍖 +2 |
| 好心旅人 | 💰 +20 |
| 废弃营地 | 免费扎营一次 |
| 晴天微风 | 本回合移动不耗体力 |

### 坏事件

| 事件 | 效果 |
|------|------|
| 沙暴来袭 | 💧-2, 🍖-1, HP-15 |
| 遭遇强盗 | 💰-25 或 HP-20 |
| 毒蝎蛰伤 | HP-25 |
| 迷路 | 体力 -1，回退上个节点 |
| 水袋破损 | 💧-3 |
| 中暑 | HP-15, 体力 -1 |

### 中性事件

| 事件 | 效果 |
|------|------|
| 废弃矿车 | 50% 💰+15 |
| 岔路发现 | 揭示隐藏捷径 |
| 沙丘瞭望 | 探查周围 2 节点 |
| 空无一物 | 无效果 |

### 概率机制

- 基础：好 40% / 中性 25% / 坏 35%
- 难度修正：简单 +10%好 / 困难 -10%好
- 节点修正：绿洲 +20%好 / 沙暴 +40%坏 / 废墟 +15%好

## 7. 跨局成长（声望系统）

成功返回营地后结算声望：

| 结算项 | 声望 |
|--------|------|
| 活着回来 | +50 |
| 带回金币（每 10） | +1 |
| 剩余水量（每点） | +2 |
| 剩余食物（每点） | +2 |
| 未用道具（每个） | +5 |
| 探索节点数（每个） | +3 |
| 难度倍率 | 普通 ×1.2 / 困难 ×1.5 |

### 可解锁内容

| 解锁项 | 声望花费 | 效果 |
|--------|----------|------|
| 🧑 探险家 | 初始 | 标准属性 |
| 👩‍🔬 地质学家 | 80 | 挖金收益 +30% |
| 🧕 游牧民 | 100 | 水/食物消耗 -20% |
| 👨‍🚀 老兵 | 120 | HP +30，战斗必胜 |
| 🛒 大背包 | 60 | 道具上限 +2 |
| 🗺️ 地图预知 | 50 | 开局显示金矿方向 |
| 🐪 老骆驼 | 70 | 免费获得骆驼 |

## 8. 技术方案

- **架构：** 纯前端单页应用，无后端
- **渲染：** Canvas 绘制节点地图 + CSS 绘制 UI 面板
- **动画：** CSS transition / keyframes
- **存档：** localStorage 存储游戏进度和声望数据
- **依赖：** 零外部依赖，总计 < 100KB
- **部署：** 静态文件，双击 index.html 或简单 HTTP 服务器
